# /value-proposition -- 가치 제안 설계

6파트 JTBD 템플릿을 사용하여 제품 또는 기능에 대한 명확하고 설득력 있는 가치 제안을 설계합니다. 고객으로부터 시작하여 실용적인 결과에 집중하는 Strategyzer의 Value Proposition Canvas의 대안입니다.

## 호출 방법

```
/value-proposition AI writing tool for non-native English speakers
/value-proposition [upload pitch deck, PRD, or competitive analysis]
/value-proposition                    # 제품에 대해 질문
```

## 워크플로우

### 1단계: 제품과 시장 이해

다음으로부터 맥락을 수용합니다:
- 제품 설명 (구두 또는 서면)
- 업로드된 문서 (피치 덱, PRD, 경쟁 분석)
- 개선할 기존 가치 제안

핵심 질문:
- 제품이 무엇을 하나요? 누구를 위한 것인가요?
- 오늘날 어떤 대안이나 임시 방편이 존재하나요?
- 어떤 고객 인사이트나 리서치를 가지고 있나요?

### 2단계: 가치 제안 구축

**value-proposition** 스킬을 적용하여 6파트 템플릿을 생성합니다:

```
## Value Proposition: [제품]

### For [세그먼트]:

1. **Who**: [타깃 사용자 프로필 및 특성]
2. **Why**: [수행하려는 작업, 원하는 결과]
3. **What Before**: [현재의 고통스러운 현실 — 기존 도구, 마찰, 임시 방편]
4. **How**: [솔루션 — 가치를 전달하는 구체적인 기능과 역량]
5. **What After**: [개선된 결과 — 가능해지는 것]
6. **Alternatives**: [당신 없이 사용할 것과 당신이 더 나은 이유]

### 가치 제안 문장
[한 문장: For [who] who [need], [product] is a [category] that [benefit]. Unlike [alternative], we [differentiator].]

### 가치 제안 문장 (재사용 가능)
- 마케팅: [...]
- 영업: [...]
- 온보딩: [...]
```

사용자에게 여러 세그먼트가 있는 경우, 각 세그먼트에 대해 별도의 가치 제안을 작성합니다.

### 3단계: 저장 및 다음 단계 제안

마크다운으로 저장합니다. 다음을 제안합니다:
- "Value Curve로 **경쟁사와 비교**할까요?"
- "이 가치 제안을 중심으로 **전체 전략을 구축**할까요?"
- "이것을 사용하여 **Lean Canvas** 또는 **Startup Canvas를 작성**할까요?"
- "이 가치 제안 문장에서 **마케팅 메시징을 생성**할까요?"

## 참고

- 이 템플릿은 고객(Who/Why)으로 시작하여 솔루션으로 향합니다 — 제품을 왼쪽에 배치하는 Strategyzer의 캔버스와 다릅니다
- 각 가치 제안은 세그먼트별 — 다른 세그먼트는 다른 가치 제안을 받습니다
- Value Curve (Blue Ocean Strategy)를 사용하여 주요 요인에 걸쳐 경쟁사와 시각적으로 제공을 비교하세요
- 가치 제안은 제품 전략의 한 요소 — 전체적인 그림을 위해 `/strategy`를 사용하세요
