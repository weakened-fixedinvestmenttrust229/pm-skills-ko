# pm-data-analytics

PM을 위한 데이터 분석 스킬: SQL 쿼리 생성 및 코호트 분석. 사용자 데이터 분석, 쿼리 생성, 리텐션 패턴 식별.

## 스킬 (3개)

- **ab-test-analysis** — 통계적 유의성, 샘플 크기 검증, 신뢰 구간, 출시/연장/중단 권고를 포함한 A/B 테스트 결과 분석.
- **cohort-analysis** — 사용자 참여 데이터에 대한 코호트 분석 수행.
- **sql-queries** — 자연어 설명으로 SQL 쿼리 생성.

## 커맨드 (3개)

- `/pm-data-analytics:analyze-cohorts` — 사용자 데이터 코호트 분석 — 리텐션 곡선, 기능 도입, 참여 트렌드.
- `/pm-data-analytics:analyze-test` — A/B 테스트 결과 분석 — 통계적 유의성, 샘플 크기 검증, 출시/연장/중단 권고.
- `/pm-data-analytics:write-query` — 자연어로 SQL 쿼리 생성 — BigQuery, PostgreSQL, MySQL 등 지원.
