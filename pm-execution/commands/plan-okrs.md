# /plan-okrs -- 팀 OKR 계획

팀 업무를 회사 전략과 연결하는 체계적인 OKR을 생성합니다. 정성적 목표와 정량적 핵심 결과를 포함한 3개의 OKR 세트를 제시합니다.

## 사용 예시

```
/plan-okrs 그로스 팀 Q2 — 회사 목표는 ARR 50% 증가
/plan-okrs "활성화율 개선"에 맞춰진 온보딩 스쿼드
/plan-okrs [회사 OKR 또는 전략 문서 업로드]
```

## 워크플로우

### 1단계: 맥락 수집

사용자에게 물어봅니다:
- 어떤 팀 또는 제품 영역을 위한 OKR인가요?
- 기간은 어떻게 되나요? (분기별이 기본이지만, 연간 또는 커스텀도 가능)
- 이에 연결되어야 할 회사 수준의 목표는 무엇인가요?
- 지난 분기는 어땠나요? (달성, 미달성, 학습)
- 제약 조건이나 알려진 우선순위가 있나요?

회사 OKR이나 전략 문서를 업로드로 수락합니다.

### 2단계: OKR 생성

**brainstorm-okrs** 스킬 적용:

- 3개의 OKR 세트 생성 (Objective + 각 3-5개의 Key Results)
- **Objectives**: 정성적, 영감을 주는, 야심차지만 달성 가능, 실행 지향적
- **Key Results**: 정량적, 측정 가능, 시간 한정, 명확한 담당자 있음
- OKR이 가시적인 연결고리와 함께 회사 목표로 연결되도록 보장
- 선행 지표(활동)와 후행 지표(결과)의 균형 유지

### 3단계: 품질 검증

각 OKR을 모범 사례와 비교하여 확인합니다:
- Objective가 영감을 주는가요? (팀을 결집시킬 수 있나요?)
- Key Results가 측정 가능한가요? (판단이 아닌 데이터로 완료 여부를 확인할 수 있나요?)
- 목표치가 야심차지만 의욕을 꺾지는 않나요? (70% 달성 = 잘 조율된 목표)
- Objective당 KR이 3-5개인가요? (그 이상 = 집중도 부족)
- KR이 어뷰징을 방지하나요? (예: "기능 5개 출시"는 쓰레기 출시를 장려함)

문제를 표시하고 개선안을 제안합니다.

### 4단계: 제시 및 반복

```
## 팀 OKR: [팀 이름] — [기간]

**연결 대상**: [회사 목표]

### Objective 1: [영감을 주는 정성적 문장]
| # | Key Result | 기준값 | 목표치 | 담당자 |
|---|-----------|----------|--------|-------|
| KR1 | [측정 가능한 결과] | [현재] | [목표] | [팀/개인] |
| KR2 | ... | ... | ... | ... |
| KR3 | ... | ... | ... | ... |

### Objective 2: [영감을 주는 정성적 문장]
[동일한 형식]

### Objective 3: [영감을 주는 정성적 문장]
[동일한 형식]

### 정렬 맵
회사 목표 → 팀 목표 → Key Results → 예상 영향

### 점수 가이드
- 0.0-0.3: 큰 미달 — 조사하고 학습
- 0.4-0.6: 진전 있으나 미달
- 0.7-0.9: 잘 조율된 스트레치 목표 — 이 범위가 목표 구간
- 1.0: 완벽히 달성했거나 목표치가 충분히 야심차지 않았음

### 체크인 주기
- **주간**: 각 KR에 대한 신호등 업데이트
- **분기 중반**: 심층 리뷰, 맥락이 변했다면 목표치 조정
- **분기 말**: 점수화, 회고, 다음 분기에 반영
```

제안합니다:
- "**목표 수준 조정**을 원하시나요 — 더/덜 공격적으로 만들까요?"
- "이것들을 추적하는 **메트릭 대시보드를 작성**해 드릴까요?"
- "이 OKR을 소개하는 **이해관계자 업데이트를 작성**해 드릴까요?"

## 참고사항

- OKR은 아웃풋이 아닌 아웃컴을 설명해야 합니다 ("온보딩 개편 출시"가 아닌 "활성화율 20% 향상")
- 사용자에게 회사 OKR이 없다면, 제품 전략이나 비즈니스 목표에서 팀 목표를 도출하도록 도와주세요
- 팀당 분기당 최대 3개의 목표 — 더 많으면 집중력이 분산됩니다
- Key Results는 스트레치 목표여야 합니다 — 달성 확신이 있다면 충분히 야심차지 않은 것입니다
- 어뷰징될 수 있는 KR을 표시하고 균형 지표를 제안하세요
