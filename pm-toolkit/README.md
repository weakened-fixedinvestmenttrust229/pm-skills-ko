# pm-toolkit

PM 유틸리티 스킬: 이력서 검토, NDA 초안 작성, 개인정보 처리방침 생성, 문법/흐름 검토. 핵심 제품 업무 이외에 제품 관리자에게 필수적인 도구들.

## 스킬 (4개)

- **draft-nda** — 두 당사자 간의 상세한 비밀유지계약서(NDA)를 초안 작성합니다.
- **grammar-check** — 텍스트의 문법, 논리, 흐름 오류를 파악하고 전체 텍스트를 재작성하지 않고 핀포인트 수정 사항을 제안합니다.
- **privacy-policy** — 데이터 유형, 관할 지역, 컴플라이언스 고려사항, 법적 검토가 필요한 조항을 포함하여 제품에 대한 상세한 개인정보 처리방침을 초안 작성합니다.
- **review-resume** — XYZ+S 공식, 키워드 최적화, 직무별 맞춤화, 구조 등 10가지 모범 사례를 기준으로 PM 이력서를 종합적으로 검토하고 개선합니다.

## 커맨드 (5개)

- `/pm-toolkit:draft-nda` — 관할 지역에 적합한 조항을 포함하여 두 당사자 간의 비밀유지계약서(NDA)를 초안 작성합니다.
- `/pm-toolkit:privacy-policy` — 데이터 수집, 이용, 저장, 컴플라이언스 요건을 포함한 개인정보 처리방침을 초안 작성합니다.
- `/pm-toolkit:proofread` — 모든 텍스트의 문법, 논리, 흐름을 검토합니다 — 전체 재작성 없이 핀포인트 수정.
- `/pm-toolkit:review-resume` — 10가지 모범 사례를 기준으로 PM 이력서를 종합 검토합니다 — 구조, 성과 지표, 키워드, 실행 가능한 피드백.
- `/pm-toolkit:tailor-resume` — PM 이력서를 특정 채용 공고에 맞춤화합니다 — 키워드 정렬, 경험 재구성, 전략적 최적화.
