# 성장 루프 (Growth Loops)

## 개요
지속 가능한 트랙션을 창출하는 성장 루프(플라이휠)를 파악하고 설계합니다. 이 스킬은 유료 유치 의존도를 줄이고 제품 주도 성장을 구축하기 위해 5가지 검증된 성장 루프 메커니즘을 평가합니다.

## 활용 시점
- 제품의 성장 메커니즘 설계
- 지속 가능한 바이럴 또는 추천 트랙션 구축
- 유료 유치 의존도 감소
- 경쟁사 성장 전략 분석
- 제품 주도 성장을 위한 제품 최적화

## 5가지 성장 루프 유형

### 1. 바이럴 루프 (Viral Loop)
사용자가 만든 제품 콘텐츠가 외부 플랫폼에 공유되어 새 사용자를 제품으로 유입시킵니다.
- **메커니즘**: 사용자가 제품 내 콘텐츠 생성 → 소셜/외부 플랫폼에 공유 → 새 사용자가 발견하고 가입
- **예시**: 링크로 공유되는 Figma 디자인, 이메일로 공유되는 Loom 영상
- **강점**: 콘텐츠가 본질적으로 공유 가능하면 기하급수적인 사용자 유치
- **과제**: 매우 공유 가능한 결과물과 공유할 강한 동기 필요

### 2. 사용 루프 (Usage Loop)
사용자가 제품 내에서 콘텐츠나 가치를 생성하고 공유하면 새 사용자를 초대하거나 재참여를 유도합니다.
- **메커니즘**: 사용자가 생성 → 창작물 공유 → 다른 사람이 소비 → 참여 사용자가 됨
- **예시**: Twitter 스레드, Medium 기사, 공개적으로 공유되는 Notion 템플릿
- **강점**: 성장이 제품 사용 및 네트워크 효과와 직접 연결
- **과제**: 콘텐츠 생성 마찰이 매우 낮아야 함

### 3. 협업 루프 (Collaboration Loop)
사용자가 제품 내에서 공동 작업을 위해 동료를 초대하여 조직 내 사용자 기반을 확장합니다.
- **메커니즘**: 사용자가 생성 → 협업을 위해 동료 초대 → 동료가 제품 가치 발견
- **예시**: Google Docs 초대, Figma 팀 프로젝트, Slack 채널
- **강점**: 깊은 조직 침투와 높은 리텐션
- **과제**: 협업/팀 기반 제품에 가장 적합

### 4. 사용자 생성 루프 (User-Generated Loop)
사용자가 다른 사용자의 창작물을 통해 새 콘텐츠나 기능을 발견하고, 자신만의 콘텐츠를 만들어 공유합니다.
- **메커니즘**: 사용자가 콘텐츠 발견 → 유사 콘텐츠 생성 → 창작물 공유 → 다른 사람이 발견
- **예시**: 크리에이터 참여를 유도하는 TikTok, Pinterest, YouTube 트렌드
- **강점**: 콘텐츠 플라이휠 및 네트워크 효과 창출
- **과제**: 지속하려면 충분한 양질의 콘텐츠가 필요

### 5. 추천 루프 (Referral Loop)
사용자가 보상, 인센티브, 또는 사회적 인정을 대가로 잠재 사용자를 초대합니다.
- **메커니즘**: 사용자가 추천 → 추천된 사용자 가입 → 추천인이 보상 획득 → 더 많이 추천
- **예시**: Dropbox 추천 보너스, Uber 라이더 추천, PayPal 가입 보너스
- **강점**: 유치를 직접 인센티브화; ROI 측정 용이
- **과제**: 단위 경제를 훼손하지 않는 가치 있는 인센티브 필요

## 작동 방식

### 1단계: 제품 가치 정의
사용자가 경험하는 핵심 가치를 명확히 합니다:
- 사용자가 제품에서 취하는 주요 행동
- 사용자 행동당 창출되는 가치
- 존재하는 네트워크 효과 (있다면)
- 경험의 마찰 포인트

### 2단계: 루프 적합성 평가
제품에 맞는 성장 루프를 평가합니다:
- 제품 유형 (협업, 콘텐츠 기반, 유틸리티 등)
- 타깃 사용자 행동 및 공유 습관
- 이미 존재하는 네트워크 효과
- 기존 사용자 기반 및 참여도

### 3단계: 루프 메커니즘 설계
구체적인 루프 구현을 만듭니다:
- 공유 또는 초대를 시작하는 트리거
- 참여 인센티브 (내재적 또는 외재적)
- 공유 메커니즘의 용이성
- 초대에서 활성화로의 전환율
- 사용자당 루프 반복 빈도

### 4단계: 루프 계수 계산
성장 속도를 추정합니다:
- 사이클당 사용자당 초대/공유 수
- 초대의 새 사용자 전환율
- 사이클당 순증 사용자
- 사이클 반복 시간

### 5단계: 루프 구축
레버리지가 가장 높은 루프를 먼저 구현합니다:
- 제품에 가장 자연스러운 루프부터 시작
- 메시징과 마찰 최적화
- 루프 지표 및 전환율 측정
- 시간이 지남에 따라 결과를 복리로 증가

## 입력 형식
$ARGUMENTS를 통해 다음을 전달합니다:
- 제품 설명 및 주요 사용자 행동
- 타깃 사용자 인구통계 및 행동
- 기존 공유/협업 기능
- 현재 성장 채널 및 지표
- 제약 조건 또는 기회

## 출력
다음을 포함하는 성장 루프 분석:
- 제품에 맞는 5가지 루프 유형의 순위 평가
- 구현 계획이 포함된 추천 주요 성장 루프
- 시간이 지남에 따라 추가할 보조 루프
- 핵심 지표 및 측정 프레임워크
- 30-60-90일 구현 로드맵
- 잠재적 루프 계수 및 성장 예측

- 협업 루프는 가장 강한 리텐션과 LTV를 만듭니다
- 최적화 단계에서 루프 건전성을 주간 단위로 측정하세요
- 규모에서 운영되면 루프를 결합하여 배수 효과를 냅니다
