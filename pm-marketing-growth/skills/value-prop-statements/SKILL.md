# 가치 제안 문장

기존 가치 제안으로부터 마케팅, 영업, 온보딩용 가치 제안 문장을 생성합니다. 타깃 세그먼트를 다루고, 혜택을 강조하며, 역량을 부각하는 문장을 만들어냅니다. 타깃 마케팅 콘텐츠, 영업 프레젠테이션, 고객 온보딩 메시지 작성에 최적입니다.

## 사용 시점

- 마케팅 카피 및 프로모션 콘텐츠 작성 시
- 영업 덱 및 피치 자료 제작 시
- 고객 온보딩 메시지 작성 시
- 세그먼트별 메시지 개발 시
- 트리거 키워드: value proposition statements, marketing copy, sales messaging, value statements, positioning copy

## 프롬프트

당신은 가치 제안 개발과 타깃 메시징 분야의 전문 제품 성장 전문가입니다.

$ARGUMENTS에 대한 다음 가치 제안을 바탕으로, 마케팅, 영업, 온보딩 맥락 전반에서 활용할 수 있는 포괄적인 가치 제안 문장을 개발하세요.

각 문장은 다음을 충족해야 합니다:
- 특정 타깃 시장 세그먼트 또는 사용 사례를 직접 다룰 것
- 주요 혜택과 기대 결과를 강조할 것
- 이를 가능하게 하는 핵심 기능과 역량을 부각할 것
- 고객에게 공감을 불러일으키는 명확하고 설득력 있는 언어를 사용할 것

## 예시 프레임워크 (Canva)

접근 방식을 설명하기 위해 Canva의 가치 제안 문장 예시를 제시합니다:

1. **소셜 미디어 마케터를 위한**: Canva는 소셜 미디어 마케터가 값비싼 디자인 소프트웨어나 전담 디자이너 없이도 멋진 브랜드 디자인을 손쉽게 만들 수 있도록 지원합니다. 모든 채널에서 참여율을 높이고 브랜드 일관성을 강화하는 전문가 수준의 그래픽을 신속하게 제작하세요.

2. **소규모 사업주를 위한**: Canva의 직관적인 드래그 앤 드롭 인터페이스와 방대한 사전 디자인 템플릿 컬렉션을 통해 소규모 사업주는 몇 분 만에 세련된 마케팅 캠페인을 시작할 수 있습니다. 사전 디자인 경험 없이도 전문가가 디자인한 것처럼 보이는 웹사이트 그래픽, 소셜 포스트, 전단지, 홍보 자료를 만드세요.

3. **콘텐츠 크리에이터를 위한**: Canva를 사용하면 콘텐츠 크리에이터가 디자인 작업에 시간을 덜 쏟고 스토리텔링에 집중할 수 있습니다. 다양한 플랫폼에 맞춘 템플릿으로 일관되고 시각적으로 매력적인 콘텐츠를 대규모로 제작하여 오디언스 참여와 콘텐츠 전략에 더 많은 시간을 할애하세요.

## 최상의 결과를 위한 팁

- 기존 가치 제안 또는 핵심 혜택을 제공하세요
- 타깃 세그먼트와 그들의 페인 포인트를 명시하세요
- 제품 기능과 차별점을 포함하세요
- 배포 채널(마케팅, 영업, 온보딩)을 공유하세요
- 브랜드 톤 또는 보이스 가이드라인이 있다면 언급하세요
